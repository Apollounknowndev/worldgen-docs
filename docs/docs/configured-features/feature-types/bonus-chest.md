## Bonus Chest

The `minecraft:bonus_chest` feature type is used to place the game's often forgotten bonus chest.

### JSON format

<pre>
{
   "type": "minecraft:bonus_chest",
   "config": {}
}
</pre>

This feature has no fields to configure.
