## Coral Claw

The `minecraft:coral_claw` feature type is used to place one type of coral. In vanilla, it is placed in the Warm Ocean biome.

### JSON format

<pre>
{
   "type": "minecraft:coral_claw",
   "config": {}
}
</pre>

This feature has no fields to configure.
