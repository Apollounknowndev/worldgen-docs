## Coral Tree

The `minecraft:coral_tree` feature type is used to place one type of coral. In vanilla, it is placed in the Warm Ocean biome.

### JSON format

```json
{
   "type": "minecraft:coral_tree",
   "config": {}
}
```

This feature has no fields to configure.
