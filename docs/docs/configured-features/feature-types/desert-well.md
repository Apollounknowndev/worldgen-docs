## Desert Well

The `minecraft:desert_well` feature type is used to place desert wells. In vanilla, it is placed in the Desert biome.

### JSON format

<pre>
{
   "type": "minecraft:desert_well",
   "config": {}
}
</pre>

This feature has no fields to configure.
