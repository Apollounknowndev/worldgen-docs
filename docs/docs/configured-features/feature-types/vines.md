## Vines

<head>
    {% include field-type-colors.html %}
</head>

The `minecraft:vines` feature type is used to place vines. In vanilla, it is used in the Jungle biome.

### JSON format

<pre>
{
   "type": "minecraft:vines",
   "config": {}
}
</pre>

This feature has no fields to configure.
